# Notas de leitura — documento DOCTORELO/DOUTORELO

O documento defende que **DOUTORELO** é uma hipótese forte para o Brasil porque une categoria médica e vínculo humano, enquanto **DOCTORELO** no exterior depende de construção semântica, pois “Elo” deixa de ser imediatamente compreensível. A leitura central é que o nome não deve ser julgado apenas por tradução, mas por capacidade de virar ativo estratégico, contrato psicológico e sistema de marca.

| Tese extraída | Implicação para o projeto |
|---|---|
| “Elo” é emocionalmente fértil em português. | O território de marca pode ser vínculo, continuidade, travessia e conexão confiável. |
| “Doctor/Doutor” aumenta confiança, mas estreita a promessa. | Se usado como marca-mãe, exige cuidado para não reduzir a plataforma a agenda médica. |
| Alternativas como DoctorFlow/DoctorLink são claras, porém genéricas. | A marca deve evitar clareza sem diferenciação e buscar propriedade semântica. |
| DOUTORELO pode colidir perceptivamente com “Doutore”. | A grafia e o sistema visual precisam separar “Doutor” de “Elo”. |
| A decisão exige arquitetura de marca. | É preciso decidir se DOUTORELO é marca-mãe, submarca ou módulo dentro de uma marca mais ampla. |
| O nome precisa carregar promessa econômica, emocional e clínica. | Branding deve integrar confiança para entrar, emoção para permanecer e clareza para comprar. |

A síntese estratégica é: **não abandonar DOUTORELO agora, mas também não canonizá-lo sem teste.** O melhor próximo passo é transformar naming e branding em um sprint de decisão, comparando hipóteses por clareza, confiança, memorabilidade, risco jurídico, elasticidade de categoria e poder visual.
