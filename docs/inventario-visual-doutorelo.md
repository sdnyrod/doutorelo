## Direção estética para os slides

A apresentação deve preservar a sensação de marca premium, médica e humana. A estética recomendada combina fundo branco/off-white, gradientes verdes suaves, cartões translúcidos, sombras discretas, círculos sobrepostos como linguagem de conexão, espaços amplos e baixa densidade textual. O resultado deve parecer uma proposta de marca para sócios, não apenas uma transcrição do PDF.
