# Padrão editorial de autoria — DOUTORELO

**Autor padrão para documentos estratégicos:** SIDNEY (PMO SENIUM.AI)  
**Aplicação:** análises estratégicas, relatórios de produto, documentos de PMO, recomendações executivas, sínteses de reuniões, interpretações de áudios dos sócios e materiais de decisão relacionados ao DOUTORELO.

Este padrão registra que documentos estratégicos que expressem a visão, a condução, a interpretação ou a governança do projeto devem ser atribuídos a **SIDNEY (PMO SENIUM.AI)**. A Manus pode atuar como ferramenta de apoio, redação, organização ou execução, mas não deve aparecer como autora desses materiais quando a entrega representar a autoria estratégica do PMO.

Quando um documento for meramente técnico, operacional ou de registro automatizado, a autoria pode ser omitida ou descrita como “documento de apoio operacional”. Em materiais executivos, a autoria deve permanecer como **SIDNEY (PMO SENIUM.AI)**, salvo orientação explícita em sentido contrário.
