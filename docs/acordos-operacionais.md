
## Entregas documentais e checkpoint

Conforme combinado com Sidney, **não é necessário salvar checkpoint para entrega de documentos**. Essa regra vale especialmente quando a entrega consiste em análise, relatório, apresentação, comparativo, proposta, documentação estratégica ou qualquer outro artefato documental que não altere produto, código ou uma versão navegável do sistema. Checkpoints devem ser reservados para mudanças de produto/código, versões de aplicação a revisar/restaurar, ou quando Sidney solicitar explicitamente.
